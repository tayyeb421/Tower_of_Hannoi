# Tower of Hanoi Game

An interactive **Tower of Hanoi** game built using **HTML, CSS, and JavaScript** to demonstrate the **Recursion** concept in **Data Structures & Algorithms (DSA)**.

## Features
-  Classic Tower of Hanoi puzzle
-  Demonstrates recursion
-  Interactive gameplay
-  Responsive design
-  Smooth animations and moves

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
